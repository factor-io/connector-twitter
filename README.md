Factor.io Twitter Connector
======================

#####visit [here](https://apps.twitter.com/), create a new application and grab these keys:
######consumer_key
######consumer_secret    
######access_token       
######access_token_secret

######You'll need all 4, and don't forget to set the read/write permissions. It defaults to just read.
